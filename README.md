# Submission Reminder App

## Description
This application reminds students about their upcoming assignment deadlines. It checks submissions and alerts students who haven't submitted.

## Directory Structure

submission_reminder_{yourName}/ ├── app/ │ └── reminder.sh ├── modules/ │ └── functions.sh ├── assets/ │ └── submissions.txt ├── config/ │ └── config.env └── startup.sh


## Installation and Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/{yourGitHubUsername}/submission_reminder_app.git

Navigate into the project folder:

cd submission_reminder_{yourName}

Run the setup script to create the necessary environment:

./create_environment.sh

Start the application:


./startup.sh
